# IW-345 — 드릴팁 이동 오류 문제

- 담당: 오윤석

## 증상
드릴팁 이동 관련 오류 — 3D 표시 위치가 실제와 불일치하거나, 이동이 예상대로 이루어지지 않음.

## 조사 포인트
- 좌표 수신 값 원본 vs 화면 표시 값
- 좌표계 변환 로직 (월드/로컬, 스케일/오프셋)
- 프리팹의 드릴팁 Transform 기준점
- 이동 보간/애니메이션 프레임 드롭 여부

## 완료 조건
- 원인 특정 + 수정
- 현장에서 실제 드릴 동작과 화면 표시 일치 확인
- 완료 시 Plane Done + 이 파일 삭제

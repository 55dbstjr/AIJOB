# Global Rules

## 언어
- 항상 한국어로 대화할 것

## 작업 환경
- 모든 작업은 Windows 기반으로 처리 (경로 구분자, 줄바꿈, 셸 명령어 등 Windows 환경 기준)

## 작업 강도
- effort는 항상 최대로 처리할 것 (최대한 깊이 사고하고, 철저히 검증하고, 충분히 탐색)

## Plane 규칙
- Plane 티켓 생성 시 담당자는 오윤석(본인)이 기본 — Member ID: `2106b5c3-0adc-495f-99a8-3ff5683a6a07`
- "티켓" = Plane, "지라 티켓"이라고 명시할 때만 Jira 사용

## 원격 서버 접속 규칙
- 192.168.3.80 서버에 SSH 접속 시 읽기 전용 명령어만 실행할 것 (ls, cat, echo, whoami, kubectl get 등)
- 원격 서버의 파일을 수정, 삭제, 생성하거나 상태를 변경하는 명령어는 실행하지 않을 것
